Installation

First, wipe disk and parition to ext4 with fdisk. Then mount the partition on /mnt and use pacstrap to install the operating system. Use arch-chroot to change the root directory to /mnt. Set a root password with passwd, then install GRUB with grub-install. Don't forget to generate a GRUB config file.

Updating

Arch Linux uses a rolling release system. Instead of freezing packages and releasing a big version of the operating system, Arch always offers the latest packages available. While this does give users the latest features, this can be a cause of system instability

Update your system with sudo pacman -Syu
